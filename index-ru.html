Знания

Владею заниями HTML на уровне кроссбраузерной верстки адаптивных веб-страниц, лендингов, шаблонов сайтов, используя возможности современных стандартов веб-технологий.

Умею использовать возможности CSS для создания стилизованных анимированных веб-страниц на основе макетов psd с точностью пиксель в пиксель.

Описание логики поведения веб-страницы с использованием принципов ООП, полный контроль элементов DOM, связь с сервером по технологиям AJAX и COMET - все это возможно с Javascript.

Я знаю как использовать адаптивную систему разметки, встроенные компоненты веб-страниц, базовые плагины JQuery для создания страниц на основе Bootstrap.

Множество плагинов, основанных на JQuery, могут быть легко подключены к разработанной веб-странице. К тому же используя эту библиотеку, можно значительно упростить и сократить код, написанный на чистом языке javascript.

Эта система управления распределенной версией помогает мне не потерять промежуточные изменений в разрабатываемом коде, позволяет писать код удаленно где бы я ни был, сотрудничать с партнерами.

Инструменты

Основным инструментом разработки, который я использую, является IDE Visual Studio Code.

Интерактивные подсказки, интегрированный git, быстрый запуск - все это ускоряет и упрощает процесс разработки в ней.

Кроме того, многие плагины позволяют настраивать ее под любые потребностям.

Облачная онлайн IDE Cloud 9 помогает мне удаленно работать через браузер.

Созданная специально для веб-разработки, она обладает всеми возможностями современных сред разработки.

Она удобно интегрирована с github и bitbucket.

Для работы с макетами формата psd я использую Adobe Photoshop CC.

Его инструменты позволяют мне верстать макеты с точностью пиксель в пиксель.

Он также позволяет обрабатывать графические элементы веб-страниц.
